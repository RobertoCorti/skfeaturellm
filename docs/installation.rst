Installation
============


Development Installation
-----------------------

To install skfeaturellm for development:

1. Clone the repository:

.. code-block:: bash

    git clone https://github.com/yourusername/skfeaturellm.git
    cd skfeaturellm

2. Install development dependencies:

.. code-block:: bash

    pip install -e ".[dev]"
    # or with Poetry
    poetry install
